# 🌏 国战·十四国天下

一个纯前端、单文件的 3D 回合制国家征服网页游戏。基于 Three.js，无后端、无打包工具、无外部资源（Three.js 走 CDN）。

打开即玩，选一个国家，征服七海。

---

## ✨ 特性

- **14 个海洋岛国**：唐、北宋、南宋、元、明、清、陈、俊、美国、俄罗斯、春秋、战国、意大利、英国。低多边形 3D 岛屿，互不接壤，海面带 sin 波浪动效。
- **3 个抽奖转盘（每 60 秒一轮，可配置）**：
  - 👤 **人物（7）**：闪速人 / 黑暗神秘人 / 黑洞之神 / 火焰人 / 超级战士 / 锁链人 / 能量之神 —— 各有攻防倍率
  - 🌀 **技能（3）**：操控别的国家 / 保护本国 / 挑拨离间
  - 🗡️ **装备（6）**：冲天弹 / 黑暗之炮 / 魔法光束 / 无中生有变出一亿军队 / 黄金铠甲 / 复活药水
- **仓库系统**：最多存 6 件物品（混存），满了自动 FIFO 丢弃最旧。
- **回合制战斗**：14 国随机顺序轮流出击。AI 自动随机攻击；轮到玩家时选目标，可消耗仓库物品。
- **军舰海运动画**：军舰沿规划路径**绕行**其它岛屿（不穿岛）→ 抛物线炮弹 → 爆炸粒子 → HP 下降。
- **战斗特写小窗（PiP）**：右下角第二相机实时显示开炮与防御画面，标注攻守双方与首领。
- **首领系统**：抽到的人物出战时作为首领，军舰上挂发光光柱 + 旋转光环（人物专属配色）。
- **音效**：Web Audio API 程序化合成（开炮 / 爆炸 / 转盘滴答 / 胜负旋律），可一键静音。
- **胜负**：玩家国家灭国 = 失败；消灭其余 13 国 = 胜利。

## ▶️ 如何运行

必须用 `http://` 协议打开（ES Modules 限制，`file://` 会失败）。在仓库根目录：

```bash
python3 -m http.server 8000
# 浏览器打开 http://localhost:8000/nationwar.html
```

> 用 `file://` 直接打开时，左上角诊断条会红色提示正确的启动命令。

---

## 🎮 操作

1. 选 14 国之一（按钮变黄高亮）→ 点「开始游戏」
2. 等待 60 秒（或倒计时到点）→ 弹出 3 个转盘 → 点「开始抽奖」→ 选保留/丢弃
3. 轮到你时 → 弹出目标选择面板 → 可选消耗仓库物品 → 发起攻击
4. HUD：我的国家 / 血量 / 军队 / 仓库 / 距下次抽奖 / 当前回合 / 音效开关

## 🛠️ 配置与调试

游戏顶部 `CONFIG` 对象可调（改源码）：

```js
spinInterval: 60,       // 抽奖间隔秒（需求是 10 分钟=600，测试用 60）
warehouseMax: 6,        // 仓库容量
startingHP: 1000,       // 起始血量
shipSpeed: 6,           // 军舰航速
worldSize: 80,          // 海洋大小
seed: 1337,             // 确定性 RNG 种子（决定岛屿布局与回合顺序）
```

浏览器 Console 调试 API：

```js
__nationwar.CONFIG.spinInterval = 5;    // 5 秒一次抽奖
__nationwar.CONFIG.warehouseMax = 99;   // 关闭仓库限制
__nationwar.spinNow();                  // 立即触发转盘
__nationwar.forceDestroy('usa');        // 立即灭 USA
__nationwar.nations.length;             // 应为 14
```
